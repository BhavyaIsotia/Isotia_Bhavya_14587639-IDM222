# Bhavya Isotia : IDM 222 Final Project

Hi! This project is my final submission for IDM 222.
It's a multi-page portfolio website built with **HTML and CSS**.

The goal of this project was to design and build a personal website that could present my work in a more complete way on a portfolio website. I wanted it to feel soft, personal, and organized while still showing a mix of UX work, development work, and creative writing!

This site demonstrates my understanding of:
- a semantic HTML structure
- CSS layout systems
- responsive design (modified my project from small screen to large screen)
- reusable styling patterns

## Project Overview

This website includes multiple pages:

*Home* - introduction, featured work, ticker(I added a slow movement thats not motion sickness causing but adds a little movement), and main navigation
*About* - personal background, interests, and image based sections (that I made swipable like a caroucel on small screens)
*Creative* - creative work, visuals, and writing projects
*Resume* - skills, experience, and resume content

Inner smaller pages:
Zara Case Study - UX research, redesign thinking, wireframes, flows, and final mockups
Constructivist Exploration - a visual style exploration based on Constructivist, Memphis, and anti-design ideas
SurfSpot Social- a web app project focused on surf forecasting, maps, and community features
Poetry Page- a writing page featuring original poems organized by themes


## Design Goals

Some of my main goals for this project were:

- create a portfolio that feels more personal than a plain resume site
- keep a soft visual theme using blush, neutral, and warm tones
- make the site feel editorial and creative without becoming messy
- organize each project page in a way that feels like a case study, not just stacked text
- make the site responsive and readable across different screen sizes
- keep the code understandable and manageable as a student project


## CSS and Layout Concepts Used

Some of the main techniques I used in this project include:

- Flexbox
- CSS Grid
- percentage widths and max-width for responsive layouts
- media queries for different screen sizes
- CSS custom properties / variables
- transitions for subtle interaction
- `overflow-x` and `scroll-snap` for swipeable image sections
- reusable card and section styling across pages
- checkbox-based dark mode toggle using CSS
